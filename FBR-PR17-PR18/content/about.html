Версия 2.0.0

📋 О приложении

Приложение для ведения заметок с полной офлайн-поддержкой и push-уведомлениями.

✨ Возможности

  • ✓ Работает офлайн (Service Worker)
  • ✓ Устанавливается на устройство (PWA)
  • ✓ Мгновенная загрузка (App Shell)
  • ✓ Push-уведомления о новых заметках
  • ✓ Реальное время (WebSocket)
  • ✓ Безопасное HTTPS соединение
  • ✓ Локальное хранение (localStorage)

📚 Выполненные практические работы

  • №13 — Service Worker (офлайн-доступ)
  • №14 — Web App Manifest (установка на устройство)
  • №15 — HTTPS + App Shell (мгновенная загрузка)
  • №16 — WebSocket + Push (реальное время и уведомления)
